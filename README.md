| Type | 내용 |
|--|--|
| solved | 해결함 |
| refactor | 리펙토링 |
| docs | 문서관련 |

```

< BLANK >
< DETAIL > 
```
```
(예시)

```

## 디렉토리 구조

* N : 해당일 수
* `본인별명`.`weekN`.`dayN` 순으로 디렉토리를 구성한다.

## 파일명

* `P` _ `문제이름` _	`문제번호`  순으로 작성한다.
예시) P_피보나치함수_1003

* 문제이름은 공백없이 붙여서 작성한다.
